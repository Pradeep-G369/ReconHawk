# Contributing to ReconHawk

Thank you for your interest in contributing!

## How to contribute

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Make your changes with clear comments
4. Test your changes against a legal target
5. Submit a pull request with a clear description

## What we welcome

- New security check modules in `core/`
- Bug fixes and performance improvements
- Dashboard UI improvements
- New report templates
- Documentation improvements

## Rules

- Only add features that work against legal targets
- Follow existing code style (comments, structure)
- Never hardcode credentials or API keys
- Test before submitting PR

## Legal

All contributions must be for defensive/authorized testing purposes only.
